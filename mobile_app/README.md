# Mobile App (React Native)

This directory will contain the React Native code for the mobile application.

## Initial File: App.js

```javascript
import React from 'react';
import { SafeAreaView, Text } from 'react-native';

const App = () => {
  return (
    <SafeAreaView>
      <Text>Hello, welcome to the Mobile App!</Text>
    </SafeAreaView>
  );
};

export default App;
```
